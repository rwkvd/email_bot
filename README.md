# Network Programmability Bot 

A lightweight Python application built as part of a Network Programmability & Automation course. 
It fetches and manages simulated network device configurations via REST API, with full CI/CD 
automation through GitLab Pipelines.

## Features
- REST API integration for network device interaction
- Automated linting and unit testing via GitLab CI/CD
- Simple UI for viewing and managing device configs
- Pipeline-driven quality control on every push

## Tech Stack
Python · REST API · GitLab CI/CD · Unit Testing
